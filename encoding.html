Encoding Methodology

<publicationStmt>

The purpose of <publicationStmt> is “to name the agency by which a resource is made available (for example, a publisher or distributor) and to supply any additional information about the way in which it is made available such as licensing conditions, identifying numbers, etc.” [2.2.4].

<publicationStmt>
    <p>Publication Information</p>
</publicationStmt>
Witness id elements

Each witness needs to be encoded with a unique @xml:id identifier so that it can be linked to variations within the text. For the printed texts the sigla used were a form of initials for the text e.g. for the 1846 Poems by Currer, Ellis and Acton Bell: <witness xml:id="ACE">

For the manuscripts I decided to adopt the sigla used by Edward Chitham in his 1979 comprehensive print edition of The Poems of Anne Brontë (updated and reissued in 2021) to facilitate easy cross-referencing of the manuscripts by readers. For example: <witness xml:id="MS-P">

This id was then used when encoding variations to indicate which readings can be found in each witness: <rdg wit="#ACE">,</rdg>
